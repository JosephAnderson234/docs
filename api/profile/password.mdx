Use this endpoint to change your own password. You must supply the **current** password for verification, plus the new password. The new password must be 8–72 characters, **different** from the current one, and follow the [password policy](/authentication#password-policy) (at least one uppercase letter, one lowercase letter, one number, and one special character).

## Endpoint

```text
PATCH /profile/password
```

## Authentication

`Authorization: Bearer <accessToken>` — **required**. Any role (`ADMINISTRADOR` or `TRABAJADOR`) may change their own password.

## Request Body

<ParamField body="currentPassword" type="string" required>
  The user's current password. Verified against the stored hash.
</ParamField>

<ParamField body="newPassword" type="string" required>
  The new password. Must be **8 to 72 characters**, **must differ** from `currentPassword`, and must follow the [password policy](/authentication#password-policy) (at least one uppercase letter, one lowercase letter, one number, and one special character).
</ParamField>

### Request example

```http
PATCH /profile/password
Authorization: Bearer <accessToken>
Content-Type: application/json
```

```json
{
  "currentPassword": "actual123",
  "newPassword": "nuevaSegura456"
}
```

## Response

### `204 No Content`

Returned on success. The response has no body. The current session remains valid and you can continue using your existing access token.

## Errors

| Status | `kind` | When |
| --- | --- | --- |
| `400 Bad Request` | `validation` | `currentPassword` is incorrect, `newPassword` equals `currentPassword`, or `newPassword` fails the [password policy](/authentication#password-policy) (length, missing uppercase, missing lowercase, missing number, missing special character, or exceeds 72 characters). |
| `401 Unauthorized` | `unauthorized` | No token provided, the token has expired, is invalid, or belongs to a session that was replaced by a newer login. |

```json
{
  "statusCode": 400,
  "error": "Bad Request",
  "kind": "validation",
  "message": "La contraseña actual es incorrecta.",
  "path": "/profile/password",
  "timestamp": "2026-06-10T10:30:00.000Z"
}

**`400 Bad Request`** — newPassword violates password policy:

```json
{
  "statusCode": 400,
  "error": "Bad Request",
  "kind": "validation",
  "message": "La contraseña debe tener al menos 8 caracteres e incluir una letra mayúscula, una letra minúscula, un número y un carácter especial.",
  "path": "/profile/password",
  "timestamp": "2026-06-10T10:30:00.000Z"
}
```
